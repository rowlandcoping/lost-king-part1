The Lost King of Catland is a portfolio project produced as part of a Level 5 Diploma in Web Development at Code Institue.

It is written using vanilla HTML, CSS and JavaScript.

The game concept borrows heavily from the Fighting Fantasy series of books, with the odd tip of the hat to games like Baldur's Gate and Disco Elysium.

Character Choices:

Select from available options to dictate how game unfolds. The decisions you make will affect the way other characters react to you - for better or for worse!

Character Statistics:

Attack: Maximum potential damage.
Skill: Chance of hitting opponent.
Defence: Potential damage prevented.
Luck: How lucky can you get?
Health: You die when this hits zero.

Items:

Some items you find in-game boost your character stats. Some... don't.

Game Objective:

You must find a way to escape the dungeon!
